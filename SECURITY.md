# Security Policy

## Supported Versions
- `main` branch (rolling)

## Reporting a Vulnerability
- Küldj emailt: security@YOURDOMAIN (amíg nincs domain: ideiglenes privát email).
- Kérünk PoC-ot, érintett verziót, hatás becslést.

## Handling
1. Bejelentés triage 24–48h.
2. Reprodukció & súlyosság (CVSS v3 durva becslés).
3. Hotfix PR (privát branch), review kötelező.
4. Release notes „Security” szekcióval.
5. Köszönet a bejelentőnek (ha kéri).

## Expectations
- Ne publikáld a hibát javítás előtt.
- Mi felelősen, gyorsan javítunk és kommunikálunk.
